# Creddy Documentation

Documentation for [Creddy](https://github.com/getcreddy/creddy) - ephemeral credentials for AI agents.

Built with [Nextra](https://nextra.site/).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Automatically deployed to Vercel on push to `main`.
