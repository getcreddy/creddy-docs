## Quick Start

```bash
# Install creddy
go install github.com/getcreddy/creddy@latest

# Start the server
creddy server

# Request a credential (from your agent)
curl -X POST http://localhost:8080/api/v1/credentials \
  -H "Content-Type: application/json" \
  -d '{"backend": "github", "ttl": "1h"}'
```
